# Autonomiczni Agenci AI z OpenClaw

> Warsztat odbywa się w formie zdalnej za pośrednictwem platformy/komunikatora online, z wykorzystaniem dźwięku, obrazu z kamery, udostępniania ekranu komputera prowadzącego i uczestników.

🟢 **Warsztat wieczorny w godz. 17:00-21:00 (4h)** 🟢

---

## Czego się nauczysz?

Podczas warsztatu uczestnicy nauczą się:

- Konfiguracji OpenClaw (najpopularniejszy projekt w historii GitHub),
- Personalizacji autonomicznego agenta działającego 24/7,
- Jak działają Heartbeats, Cron, Skills, Memory, Web Search,
- Podłączenia komunikatora (np. Whatsapp),
- Podstaw bezpieczeństwa

Warsztat skierowany do:

## Wymagania wstępne

- **Podstawowa znajomość terminala** (CLI, TUI, np, doświadczenie z Claude Code),
- **Dedykowane urządzenie dla Agenta**, np. stary laptop (najlepiej Linux lub macOS, ale Windows też ok), macMini / mini PC - opcje z Docker i/lub VPS nie bedą omawiane na tym szkoleniu, ale możliwe),
- **Dostęp do LLM** (najlepiej Opus 4.6 i/lub GPT-5.3-Codex) przez: API (OpenAI, Anthropic, Azure, AWS, GCP, GLM, itp.),
- **Subskrypcję** (ChatGPT z Codex CLI, Claude Code, Z.ai Coding Plan - lokalne modele nie pokażą pełni możliwości, ale model 30b+ z dobrym function calling da podstawową wartość),
- **Konto Tailscale** (darmowe),
- **Zainstalowany Node.js** z npm oraz Git,
- **API wyszukiwarki** (najlepiej Perplexity lub Grok Search, ale Brave darmowy też działa),
- **Komunikator prywatny** (np. Whatsapp) lub firmowy z dostępem admina (Slack, Teams),
- **Chęć eksperymentowania i podejmowania ryzyka.**
